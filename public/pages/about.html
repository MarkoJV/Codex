About The Codex

The Codex is made for KymenlaaksoCodeBootcamp as a student demo app. Main points were to learn about asynchronous code, Progressive Web Apps, Service Workers and Google Firebase usage. This project was created for learning purposes. Project is not feature complete.

The Codex was made using Javascript/HTML and Materialize css library. Database and hosting by Google FireBase. Application is designed to meet all PWA standards and has limited offline functionality. Much of the learning was made by following tutorials by The Net Ninja.
